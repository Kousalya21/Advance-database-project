Python version used: Python 3.7

import the csv file all_months.csv in your AWS database name the table as 'quakes'

In awsdb.py:
1. replace the connection variables with your own AWS database connection variables.

In awsredis.py:
1. replace the connection variables with your own AWS Redis connection variables.

To run locally, run the following commands:

pip install -r requirements
python application.py

Use the file 'HTTP Request.jmx' with jmeter for load testing
